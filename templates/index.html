Oracle LP math

This is the optimization problem the oracle will solve later. The machine knows the real CSV load/PV data and chooses the unknown battery actions that minimize the bill.

Known parameters

Load(t)
Building load from the CSV, in kW.
PV(t)
Solar generation from the CSV, in kW.
BatteryCapacity
Battery energy capacity, in kWh.
BatteryPowerLimit
Maximum battery charge/discharge power, in kW.
ChargeEfficiency
Fraction of charging energy that becomes stored energy.
DischargeEfficiency
Fraction of stored energy that comes back out usefully.
dt
Time step in hours. Current default is 0.25 h.
EnergyPrice(t)
Electricity tariff at time t, in VND/kWh.
DemandChargeRate
Capacity charge, in VND/kW.
BatteryWearCost
Battery degradation cost, in VND/kWh throughput.
MinimumSOC
Lowest allowed battery state of charge.
MaximumSOC
Highest allowed battery state of charge.
RequiredFinalSOC
Minimum SOC required at the end of the horizon.

Unknown variables to solve

BatteryDischarge(t)
Battery to load power, in kW.
GridCharge(t)
Grid to battery power, in kW.
SolarCharge(t)
PV surplus to battery power, in kW.
GridImport(t)
Power imported from the grid, in kW.
BatterySOC(t)
Battery state of charge from 0 to 1.
PeakGrid
Monthly billed peak demand, in kW.

Objective

MINIMIZE:

  Σ[t=1→96] (
      EnergyPrice(t) × dt × (
          GridCharge(t) - BatteryDischarge(t)
      )
  )
      ← Net energy cost

+ Σ[t=1→96] (
      BatteryWearCost × dt × (
          BatteryDischarge(t)
        + GridCharge(t)
        + SolarCharge(t)
      )
  )
      ← Battery getting old

+ DemandChargeRate × PeakGrid
      ← Monthly demand charge

Constraints

GridImport(t)
  = EffectiveLoad(t) + GridCharge(t) - BatteryDischarge(t)
  ≥ 0
      ← No electricity export

BatterySOC(t+1)
  = BatterySOC(t)
  + (ChargeEfficiency × dt / BatteryCapacity) × GridCharge(t)
  + (ChargeEfficiency × dt / BatteryCapacity) × SolarCharge(t)
  - (dt / (DischargeEfficiency × BatteryCapacity)) × BatteryDischarge(t)
      ← Battery SOC update

0 ≤ BatteryDischarge(t) ≤ BatteryPowerLimit
0 ≤ GridCharge(t) ≤ BatteryPowerLimit
0 ≤ SolarCharge(t) ≤ BatteryPowerLimit
0 ≤ SolarCharge(t) ≤ SolarSurplus(t)

MinimumSOC ≤ BatterySOC(t) ≤ MaximumSOC
BatterySOC(96) ≥ RequiredFinalSOC

0.5 × (GridImport(t) + GridImport(t+1)) ≤ PeakGrid
for t = 0 ... 95
      ← 30-minute rolling average demand
